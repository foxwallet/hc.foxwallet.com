# 介绍
FoxWallet 是一款安全好用的 去中心化 Web3 钱包，致力于成为 Web3 世界的入口和连接器。

FoxWallet 开发团队拥有丰富的区块链和网络安全从业经验，代码通过了权威审计机构的[审计](https://github.com/foxwallet/security-audit-certification)，是值得信赖的靠谱钱包。
