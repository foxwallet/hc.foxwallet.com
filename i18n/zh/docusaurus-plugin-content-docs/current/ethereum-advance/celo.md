# Celo

[FoxWallet](https://foxwallet.com)是最好用的Celo钱包。

## 添加 Celo

“设置” => “管理 ETH RPC” => 勾选 Celo => 返回到钱包主页。

![](../img/add-celo.webp)

## 切换至 Celo

点击钱包主页左上角的切换按钮 => 选择 Celo => 选择一个账户。

![](../img/switch-celo.webp)
