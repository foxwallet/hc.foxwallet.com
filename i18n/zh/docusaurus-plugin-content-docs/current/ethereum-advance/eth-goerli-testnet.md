# 以太坊测试网 Goerli (Görli)

## 添加网络
打开FoxWallet，进入设置页 => "管理 ETH RPC" => 勾选 Goerli Testnet => 返回到钱包主页。

![](../img/manage-eth-rpc.webp)![](../img/add-goerli.webp)

## 切换网络
点击左上角的切换按钮 => 选择 Goerli Testnet => 选择一个账户。

![](../img/switch-network.webp)![](../img/switch-goerli.webp)
