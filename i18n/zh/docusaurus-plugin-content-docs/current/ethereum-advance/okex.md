# OKC (OKXChain、OKExChain)

FoxWallet是最好用的OKC钱包，OKXChain钱包，OKExChain钱包，OKT钱包，OK链钱包。

## 添加 OKC (OKExChain)

“设置” => “管理 ETH RPC” => 勾选 OKC (OKExChain) => 返回到钱包主页。

![](../img/add-okex.png)

## 切换至 OKC (OKExChain)

点击钱包主页左上角的切换按钮 => 选择 OKC (OKExChain) => 选择一个账户。

![](../img/switch-okex.png)
