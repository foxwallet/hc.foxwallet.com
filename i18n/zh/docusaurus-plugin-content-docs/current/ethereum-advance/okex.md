# OKEX Chain

FoxWallet是最好用的OKEX Chain钱包，OKT钱包，OK链钱包。

## 添加 OKEX Chain

“设置” => “管理 ETH RPC” => 勾选 OKEX Chain => 返回到钱包主页。

![](../img/add-okex.png)

## 切换至 OKEX Chain

点击钱包主页左上角的切换按钮 => 选择 OKEX Chain => 选择一个账户。

![](../img/switch-okex.png)
