# Moonbeam

FoxWallet是最好用的Moonbeam钱包，最好用的GLMR钱包。

## 添加 Moonbeam

“设置” => “管理 ETH RPC” => 勾选 Moonbeam => 返回到钱包主页。

![](../img/manage-eth-rpc.webp)![](../img/add-moonbeam.webp)

## 切换至 Moonbeam

点击钱包主页左上角的切换按钮 => 选择 Moonbeam => 选择一个账户。

![](../img/switch-network.webp)![](../img/switch-moonbeam.webp)
