## 用法
- 下载并打开[FoxWallet](https://foxwallet.com/download)

- 进入 “我” -> “管理网络” 选择并下载 “Aleo" 的零知识证明参数以供使用。

- 返回钱包首页，点击左上角切换按钮，[切换网络](https://hc.foxwallet.com/zh/docs/basic/manage-funds#switch-networks) 到`Aleo`
