# KCC

[FoxWallet](https://foxwallet.com) 最安全好用的KuCoin社区链钱包, 最好用的KCC钱包, 最好用的KCS钱包。

## 添加 KCC

“我” => “管理网络” => 勾选 KCC Mainnet => 返回到钱包主页。

![](../img/manage-eth-rpc.webp)![](../img/add-kcc.webp)

## 切换到 KCC

点击钱包主页左上角的切换按钮 => 选择 KCC Mainnet => 选择一个账户。
