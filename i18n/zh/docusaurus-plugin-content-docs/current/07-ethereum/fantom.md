# Fantom

[FoxWallet](https://foxwallet.com)是最好用的Fantom钱包，最好用的FTM钱包。

## 添加 Fantom

“我” => “管理网络” => 勾选 Fantom => 返回到钱包主页。

## 切换至 Fantom

点击钱包主页左上角的切换按钮 => 选择 Fantom => 选择一个账户。
