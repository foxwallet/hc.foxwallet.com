# Conflux eSpace

[FoxWallet](https://foxwallet.com)是最好用的Conflux钱包，最好用的CFX钱包。

## 添加 Conflux eSpace

“我” => “管理网络” => 勾选 Conflux eSpace => 返回到钱包主页。

## 切换至 Conflux eSpace

点击钱包主页左上角的切换按钮 => 选择 Conflux eSpace => 选择一个账户。
