# Arbitrum One

[FoxWallet](https://foxwallet.com) 是最好用的Arbitrum钱包，最好用的layer2钱包。

## 添加 Arbitrum One

“我” => “管理网络” => 勾选 Arbitrum One => 返回到钱包主页。

## 切换至 Arbitrum One

点击钱包主页左上角的切换按钮 => 选择 Arbitrum One => 选择一个账户。
