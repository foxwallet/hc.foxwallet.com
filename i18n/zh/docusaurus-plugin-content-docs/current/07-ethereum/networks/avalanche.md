# Avalanche C-Chain

[FoxWallet](https://foxwallet.com)是最好用的Avalanche钱包，最好用的AVAX钱包。

## 添加 Avalanche C-Chain

“我” => “管理网络” => 勾选 Avalanche C-Chain => 返回到钱包主页。

## 切换至 Avalanche C-Chain

点击钱包主页左上角的切换按钮 => 选择 Avalanche C-Chain => 选择一个账户。
