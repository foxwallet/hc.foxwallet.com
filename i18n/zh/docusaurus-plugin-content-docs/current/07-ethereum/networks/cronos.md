# Cronos Mainnet Beta

[FoxWallet](https://foxwallet.com)是最好用的Cronos钱包，最好用的CRO钱包。

## 添加 Cronos Mainnet Beta

“我” => “管理网络” => 勾选 Cronos Mainnet Beta => 返回到钱包主页。

## 切换至 Cronos Mainnet Beta

点击钱包主页左上角的切换按钮 => 选择 Cronos Mainnet Beta => 选择一个账户。
