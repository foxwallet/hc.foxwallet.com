# Optimism

[FoxWallet](https://foxwallet.com)是最好用的Optimism钱包，最好用的layer2钱包。

## 添加 Optimism

“我” => “管理网络” => 勾选 Optimism => 返回到钱包主页。

## 切换至 Optimism

点击钱包主页左上角的切换按钮 => 选择 Optimism => 选择一个账户。
