# Aurora

FoxWallet is the best Aurora wallet.

## Add Aurora

“Setting” => “Manage ETH RPC” => Enable Aurora => Back to wallet main page.

![](../img/add-aurora.png)

## Switch to Aurora

Click the switch button in main page => Choose Aurora => Select one
account.

![](../img/switch-aurora.png)
