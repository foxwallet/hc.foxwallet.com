# BNB Chain (BSC)

## Add BNB Chain

“Setting” => “Manage ETH RPC” => Enable BNB Chain => Back to wallet main page.

![](../img/add-bsc.png)

## Switch to BNB Chain

Click the switch button in main page => Choose BNB Chain => Select one
account.

![](../img/switch-bsc.png)
