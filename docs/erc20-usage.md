# ERC20 Usage

## Add ERC20 Token

Click the add button in main page => Input the token symbol or contract address to search => Click the “+” button.

![](./img/add-erc20.png)

## Remove ERC20 Token

Click the add button in main page => Click the “-” button on the token you want to remove。

![](./img/rm-erc20.png)
